### Q. Describe primitive data types in Java. (4 Marks)

Java provides **8 primitive data types** that store simple values and are not objects. These types are built into the language and used for basic data handling and memory-efficient operations.

**Primitive Data Types:**
- **byte (1 byte):** Stores small integers (â€“128 to 127).
- **short (2 bytes):** Stores medium range integers.
- **int (4 bytes):** Default integer type, widely used.
- **long (8 bytes):** Stores large integer values.
- **float (4 bytes):** Stores decimal numbers (single precision).
- **double (8 bytes):** Stores decimal numbers (double precision, default for floating values).
- **char (2 bytes):** Stores a single Unicode character.
- **boolean (1 bit):** Stores true or false values.

Primitive types are **faster**, memory-efficient, and directly supported by the JVM.
