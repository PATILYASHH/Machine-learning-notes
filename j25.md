### Q. Explain use of `final` keyword in Java. (4 Marks)

The `final` keyword in Java is used to **restrict modification**. It can be applied to variables, methods, and classes.

---

#### ✅ `final` variable
- Value cannot be changed once assigned (constant).

**Example:**
```java
final int x = 10;
// x = 20; // Error: cannot change value


---

✅ final method

Cannot be overridden by subclasses.


Example:

class A {
    final void show() {
        System.out.println("Final method");
    }
}


---

✅ final class

Cannot be inherited (no subclass allowed).


Example:

final class Test { }
// class Demo extends Test { } // Error


---

The final keyword ensures security, immutability, and controlled behavior in Java programs.
