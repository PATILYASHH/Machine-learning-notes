ğŸ› Troubleshooting

"Command not found: python3"

Solution: Try python instead:

python --version
python build.py

"No module named 'markdown'"

Solution: Install dependencies:

pip install markdown Pygments
# or
pip install -r requirements.txt

"Address already in use"

Solution: Port 8000 is taken, use different port:

python3 -m http.server 8001

Then visit http://localhost:8001

"Permission denied"

Solution: Use sudo or different port:

python3 -m http.server 8080

Website loads but no styling

Solution: Check that static folder exists:

ls site/static/
# Should show: script.js  style.css

If missing, rebuild:

python3 build.py

Bookmarks not saving

Solution: - Check browser console for errors (F12) - Try a different browser - Clear browser cache

Mobile view not working

Solution: - Clear browser cache - Try incognito/private mode - Check responsive design mode (F12 â†’ Toggle device toolbar)

cd site && python3 -m http.server 8000

# When done, deactivate
deactivate
