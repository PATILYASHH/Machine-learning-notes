What is Data Preprocessing? Why Do We Need It? (4 Marks)

Definition:
Data preprocessing is the initial step in a Machine Learning workflow where raw data is cleaned, transformed, and prepared before feeding it to a model.
It helps convert messy or incomplete data into a structured and meaningful format suitable for training.

Why Do We Need Data Preprocessing?

  1. Removes Noise and Errors
  2. Real-world data often contains irrelevant or incorrect information.

  3. Preprocessing removes such noise for better learning accuracy.

  4. Handles Missing Values

  5. Missing or incomplete data can lead to wrong model predictions.

  6. Preprocessing fills or removes missing entries appropriately.

  7. Ensures Consistency

  8. Data may come from different sources or formats.

  9. Preprocessing makes it uniform and consistent across all samples.

  10. Improves Model Performance

  11. Clean and normalized data helps ML algorithms train faster and predict more accurately.

  12. Prepares for Feature Scaling

  13. Normalization or standardization ensures all features are on the same scale, improving algorithm performance (especially for distance-based models like KNN or SVM).

✅ In short:
Data preprocessing is essential to convert raw data into high-quality input, reducing noise and improving the accuracy, efficiency, and reliability of Machine Learning models.
