### Q. Difference between `this` and `super` keyword. (4 Marks)

| `this` Keyword | `super` Keyword |
|----------------|----------------|
| Refers to the **current class object** | Refers to the **parent class object** |
| Used to access **current class variables and methods** | Used to access **parent class variables and methods** |
| Used to call **current class constructor** | Used to call **parent class constructor** |
| Resolves **name conflict** between instance variables and parameters | Resolves **overridden members** in child class |
| Cannot access parent class members | Cannot access current class members |

```java
class Parent {
    int x = 10;
}

class Child extends Parent {
    int x = 20;

    void show() {
        System.out.println(this.x);  // current class variable -> 20
        System.out.println(super.x); // parent class variable -> 10
    }
}
