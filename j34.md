### Q. Explain how to define and use user-defined package with example. (8 Marks)

A **user-defined package** in Java is a custom package created by the programmer to organize classes logically and avoid name conflicts.  
Packages help in **modular programming, code reuse, and better maintenance**.

---

### ✅ Steps to Create & Use User-Defined Package

#### **1. Create a Package**
Use the `package` keyword at the top of the file.

#### **2. Compile the file**
Use `javac -d . filename.java` to create package folders automatically.

#### **3. Import and use the package**
Use the `import` keyword in another program.

---

### ✅ Example

#### **File 1: Create Package & Class**  
**Path:** `mypack/Message.java`
```java
package mypack;

public class Message {
    public void show() {
        System.out.println("Hello from user-defined package!");
    }
}

Compile:

javac -d . Message.java


---

File 2: Use the Package

Main program

import mypack.Message;

public class TestPackage {
    public static void main(String[] args) {
        Message m = new Message();
        m.show();
    }
}

Run:

java TestPackage


---

✅ Output

Hello from user-defined package!


---

User-defined packages allow us to organize code, avoid naming conflicts, and improve clarity and modularity in large Java projects.
